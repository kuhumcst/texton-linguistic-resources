* https://github.com/dsldk/middelaldertekster/tree/master/data. Not all files in this directory are lemmatized. Those that are have indices into lemma lists go_lemmaliste.csv and ko_lemmaliste.csv, which were kindly provided by Nicolai Hartvig Sørensen 15 May 2018
* https://github.com/Clara-Kloster/Guldkorpus
